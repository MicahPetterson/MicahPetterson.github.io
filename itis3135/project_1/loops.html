Loops

Ah, loops. Now these are pretty fun, I'm not gonna lie. Loops do exactly what you think they do when you first hear the term, they loop! Now they don't just loop on forever but you will see in a minute what I am talking about. There are two basic types of loops in C, for loops and while loops. There are pretty distinct differences between the two and they should be used in different circumstances. We are going to start off talking about for loops.

For Loops

For Loops are the loops that you are probably going to see the most of early on as they are used for lots of things. Loops may sound like a thing that is not very important but you will soon find out just how important they are and how much you need them to do so many things. Let's start out with the basic setup of a for loop. I am going to provide the code skeleton first and then explain the different parts later.

for (iterator variable; condition; increment iterator)

{

loop content;

}

Here is one filled out for you,

for (int i = 0; i < 100; i++)

{

int sum = sum + 10;

}

Now, you may be wondering what these are actually useful for. Well one of the major uses is to iterate through (or look through) an array. We have the ability to view whatever is in each spot of an array and do whatever we want with that data. For example, let's assume we have an array of integers named numbers and it already has multiple values inside of it. Imagine you wanted to take each one of these numbers and add them together to get the sum of all of the integers in numbers. This is what it might look like,

int sum = 0;

for (int i = 0; i < sizeOf(numbers); i++)

{

sum += number[i];

}

To give you an understanding of how this might work, type any positive whole number in the blank below. Then, click the "Run" button, and you will see the sum of all the numbers from 0 to the number you entered, all thanks to this for loop!.

int yourNumber = ;

int sum = 0;

for (int i = 0; i <= yourNumber; i++)

{

sum += i;

printf("Sum: %d", sum);

In the previous function, we are iterating through the array by putting 'i' inside the brackets next to the array name. Since we are trying to sum all of the numbers in the array together we need to continue adding each number to a variable named 'sum'. This will iterate through the entire length of the array because as part of the conditional we stated that the loop will keep going while 'i' is still less than the size of the array. If you are wondering how we did that, it was the sizeOf() function which tells the computer the size of the array that we put in it.

Now that you know the fundamentals of the For loop, we are going to talk about While loops.

While Loops

So you might be wondering why there are different types of loops and what the different uses of these might be. For loops are very useful when you already know the amount of times you need to loop, and while loops are useful when you don't know the amount of times you need to loop. Now this will not really be a need for you so early in your programming career but it is still useful knowledge to know how to build and use a while loop. Here is the basic outline for a while loop:

while (controller)

{

(statement to loop through);

}

Now obviously this example doesn't show you exactly how it might work so let's explore it a little more. The controller statement is what allows the loop to actually run. As long as the controller statement is true then the loop will continue to run. Let's look at another example:

int i = 0;

while (i < 5)

{

printf(“i is %d”, i);

i++;

}

Now the loop I just showed you could be accomplished as a for loop but that's okay for now. You really just need to know these two types of loops for now.

One of the major differences between for loops and while loops that we haven't talked about yet is when the loops check the condition statement. For loops check the condition statement before it executes the statement inside the loop, meanwhile while loops check the condition after the statement is executed. This means that even if the condition is false a while loop will always run at least once. Now lets see if you can apply this knowledge by looking at this example:

int x = 5;

for (int i = 0; i > 5; i++)

{

System.out.println(“This loop is running”);

}

int counter = 0;

while (counter > x)

{

System.out.println(“No, this loop is running”);

}

Now what will this program print out?

This loop is running

This loop is running

No, this loop is running

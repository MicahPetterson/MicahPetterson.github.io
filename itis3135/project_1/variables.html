Variables

Oh variables, one of the most basic building blocks of programming. Variables are what we use to hold data in a program, we can use variables to manipulate this data in many different ways. Whether it be to do math using them, use them to store data that the user inputs, and many other things. As you will see, variables are very very very important to every programming language and is how we do anything productive in programs. First we will start with how to initialize, which means to give a variable data, or to put it into the program that you are writing.

Say we have a variable that we want to name “number” that we will use to store the number 5 in a program.

int number = 5;

You will see a couple of things that we haven't mentioned yet, “int” and the semicolon at the end of the statement. We will start by discussing what “int” means and what other words we can use when we are declaring variables. We call these words data types and this table will show you the different data types in C and what data can be stored in them.

The semicolon that we discussed in the last paragraph is what is used to end every statement in C. It is very important that they are used to end every line because it is what tells the computer that a statement is finished, so it will not run without any semicolons.

Printing

We are also going to talk about printing on this page. Now I'm not gonna lie, printing is a little bit more annoying in C than in many other languages, but it's not that bad. Now you may be asking, “What is printing?”. Good question! It is something we do when we want to print the output to where we can see it. So that may be the console, the console inside of an IDE, etc. but whatever is being printed will be seen by the user. This is the code we would use to print something out:

("Hello World!");

This would print out:

Hello World!

Printing is very useful in programming as it allows the user to actually be involved with the code that has been written. It can be more tricky when you want to print out a variable because a little bit more is needed. In order to add a variable to a print statement, we need to specify its data type using the “%” operator followed by a letter telling the computer what kind of variable it is. One more thing needs to be added, after the quotes you will add a comma followed by the variable name. Like so:

int x = 5;

printf(“This number is %d”,x);

Here are the symbols for the different data types:

Data Type Symbol
char %c
int %d or %i

string %s

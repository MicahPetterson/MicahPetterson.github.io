Project 1 Proposal

Project Overview

Throughout the course of this project, I will be developing for my client a website intended to teach students interesting in learning programming how to do so in the C programming language. My client will provide me with the necessary information to create a website that will teach users the basics of C programming from a non-computer science student perspective. The pages of the site will cover basic topics such as variables, pointers, and functions, much of which will be interactive to give the user a hands-on experience in further understand how code works.

Site Map

Page Design

  • Home - Contains introductory information on the programming language C, intended to break down the idea of programming and the purpose of this site for those visiting it for the first time, and presumably those who are less experienced with computer science in general. This page will not have any interactive elements, intended solely to introduce the learning user to the rest of the site.
  • Variables - This page will explain the concept of variables in computer programming, serving as the preface page to the subsequent child pages that will discuss each individual variable type. This page (like all of the pages on this site) will be intended to educate the viewer on the subject it is named after. In this case, C variables and what they are used for. The Variables page and all of its child pages in the sublist below will not contain interactive elements, simply explaining the use of variables while providing examples.
    • char - Will explain the use of char type variables in C programming, and what they are used for.
    • int - Will explain the use of int type variables in C programming, and what they are used for.
    • float - Will explain the use of float type variables in C programming, and what they are used for.
    • string - Will explain the use of strings in C programming, and what they are used for.
  • Pointers - This page will explain the concept of pointers in C programming, and how they work in practice. The page will include interactive elements to provide the user a hands-on experience in learning how pointers function.
  • Loops - This page will introduce the various types of loops used in C programming, and how each one operates. This webpage will feature interactive elements that display the functionality of each loop, and how they differ from one another in their respective iterations.
  • Functions - In this page, the user will be provided with information on how functions work in C programming. interactive elements will also be used here in order to help students understand how parameters, return values, and other aspects of functions work in this language.

Dynamic Functionality

Throughout this website, JavaScript will be utilized to allow users to modify the code in faux C programs, allowing them to experiment with code to see how it will affect the output. Users will be allowed to change variable values, types, pointers, and other aspects of the code to see how it will affect the execution.
